Benchmarks were run on the local development machine after installing FFmpeg with Homebrew.

| Tool | Version |
| --- | --- |
| macOS / Darwin | 25.5.0 |

| FFmpeg | 8.1.2 |

FFmpeg was installed from Homebrew core with H.264, H.265, AV1, VideoToolbox, AudioToolbox, AAC, Opus, and related support enabled.

## Results

| Check | Command Shape | Result |
| --- | --- | --- |
| Metadata probe | `ffprobe -show_streams -show_entries format` | Passed in `0.02s`; detected 1280x720 H.264 video, 30 FPS, 180 frames, 6.0s duration, AAC mono audio, 2.1 MB file size. |
| Thumbnail extraction | `ffmpeg -ss 00:00:02 -frames:v 1` | Passed in `0.05s`; wrote `phase0-thumbnail.jpg`. |
| Waveform image | `ffmpeg -filter_complex showwavespic` | Passed in `0.03s`; wrote `phase0-waveform.png`. |
| Silence detection | `ffmpeg -af silencedetect=noise=-50dB:d=0.5 -f null` | Passed in `0.03s`; detected silence from `0.0s` to `1.500021s`. |
| Audio enhancement filter chain | `highpass`, `lowpass`, `acompressor`, `loudnorm`, `aresample` | Passed in `0.11s`; wrote 48 kHz mono PCM WAV output. |
| Vertical export smoke test | scale and pad to 1080x1920, H.264/AAC | Passed in `1.52s`; output is 1080x1920, 30 FPS, 6.016s container duration, about 950 KB. |
| YouTube-style export smoke test | H.264/AAC transcode | Passed in `0.96s`; output is 1280x720, 30 FPS, 6.016s container duration, about 2.0 MB. |

## Observations

- FFprobe metadata extraction is fast enough for synchronous probing of individual files, but Phase 1 should still run batch imports as background jobs to protect UI responsiveness.
- Thumbnail and waveform generation are fast on the synthetic fixtures. Real camera media, long files, and high-bitrate codecs still need broader tests in Phase 2.
- The FFmpeg audio filter chain is viable for MVP-style enhancement. It should be wrapped in explicit presets and should preserve expected sample rates.
- Export smoke tests confirm FFmpeg can cover the planned social format paths. Phase 6 should add progress parsing and cancellation through Rust job management.
- The installed FFmpeg build includes GPL-enabled codecs such as x264. Release packaging must review codec and license implications before distribution.
